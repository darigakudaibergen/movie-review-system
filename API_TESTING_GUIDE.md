# API TESTING GUIDE

## API Verification Summary

The Movie Review System provides a REST API for managing movies, genres and reviews.

The following endpoints are available and were verified locally:

### Genres API
- GET /api/genres/
- POST /api/genres/
- DELETE /api/genres/{id}/

### Movies API
- GET /api/movies/
- GET /api/movies/{id}/
- POST /api/movies/
- PUT /api/movies/{id}/
- DELETE /api/movies/{id}/

### Reviews API
- GET /api/reviews/
- GET /api/reviews/{id}/
- POST /api/reviews/
- PUT /api/reviews/{id}/
- DELETE /api/reviews/{id}/

API endpoints were tested locally.
Movies API supports GET, POST and DELETE operations.
